一、文件上传规则
========================================

1.1 支持的文件格式
- 图片格式：JPG, JPEG, PNG, GIF, BMP, WEBP, SVG
- 文档格式：PDF, DOC, DOCX, XLS, XLSX, PPT, PPTX, TXT, MD
- 视频格式：MP4, AVI, MOV, MKV, WMV, FLV
- 音频格式：MP3, WAV, AAC, FLAC, OGG
- 压缩格式：ZIP, RAR, 7Z
- 其他格式：所有常见文件格式均支持

1.2 文件命名规范
- 文件名长度不超过255个字符
- 支持中文、英文、数字及常用符号
- 不建议使用特殊字符：\ / : * ? " < > |
- 建议使用有意义的文件名，便于查找和管理

1.3 上传方式
- 点击上传：点击"上传文件"按钮选择文件
- 拖拽上传：将文件拖拽到上传按钮区域
- 支持同时上传多个文件
- 上传按钮在拖拽时会有视觉反馈

1.4 文件存储策略
- 小文件（≤5MB）：使用localStorage本地存储
  - 优点：上传速度快，无需网络连接
  - 限制：受浏览器localStorage配额限制
  
- 大文件（>5MB）：自动使用MEGA云存储
  - 优点：存储空间大，支持大文件
  - 特点：上传时显示"上传到Mega中..."提示
  - 文件链接永久保存，可随时访问

1.5 最大单个文件大小限制
- 单个文件最大限制：无硬性限制
- 小文件受localStorage配额限制（通常为5-10MB
- 大文件使用MEGA云存储，支持更大文件
- 建议超大文件建议分卷压缩后上传

========================================

二、MEGA云存储使用说明
========================================

2.1 MEGA存储特点
- 自动切换：系统自动根据文件大小选择存储方式
- 无缝集成：用户无需手动选择，系统自动处理
- 安全可靠：使用MEGA官方API，数据安全有保障
- 大容量：支持大文件上传，突破localStorage限制

2.2 MEGA上传流程
1. 上传大于5MB的文件
2. 系统自动检测文件大小
3. 显示"上传到Mega中..."状态提示
4. 等待上传完成
5. 文件自动保存到文件列表
6. 文件标记为MEGA文件（isMegaFile: true）

2.3 MEGA文件标识
- MEGA文件与本地文件在文件列表中显示一致
- 系统内部通过isMegaFile属性区分
- 用户操作方式与本地文件相同
- 下载、预览、删除等功能完全兼容

2.4 MEGA存储注意事项
- 需要稳定的网络连接
- 大文件上传时间较长，请耐心等待
- 上传过程中请勿关闭浏览器标签页
- 如遇上传失败，请检查网络连接后重试

========================================

三、账户存储空间管理
========================================

3.1 存储空间限制
- 每个账户总存储空间：5 GB
- 存储空间包含所有文件（含回收站文件）
- 已删除文件在回收站中仍占用空间
- MEGA存储不计入localStorage配额，但计入总存储空间

3.2 已使用空间查询方法
- 登录后在左侧边栏查看存储空间使用情况
- 显示已使用空间和总空间百分比
- 彩色进度条直观展示空间使用状态

3.3 空间不足时的提示机制
- 当存储空间使用超过90%时，系统将显示警告提示
- 当存储空间使用超过95%时，系统将显示紧急提示
- 建议及时清理不需要的文件或考虑升级存储空间

3.4 扩容途径
- 当前版本暂不支持手动扩容
- 如需更多空间，请联系管理员
- 后续版本将推出付费扩容功能

四、操作指南与流程说明
========================================

4.1 账户注册
- 当前版本使用预设账户登录
- 管理员账户：admin@example.com
- 管理员密码：admin123
- 后续版本将开放用户自助注册功能

4.2 登录流程
1. 访问网站首页
2. 输入邮箱地址
3. 输入密码
4. 点击"登录"按钮
5. 登录成功后自动跳转至主页

4.3 文件上传流程
方法一：拖拽上传
1. 在主页空白处拖拽文件
2. 系统自动判断文件大小
3. 小文件：直接上传到localStorage
4. 大文件：显示"上传到Mega中..."并上传到MEGA
5. 等待上传进度条完成
6. 文件自动显示在文件列表中

方法二：点击上传
1. 点击"上传文件"按钮
2. 在弹出的文件选择框中选择文件
3. 系统自动判断文件大小
4. 小文件：直接上传到localStorage
5. 大文件：显示"上传到Mega中..."并上传到MEGA
6. 等待上传进度条完成
7. 文件自动显示在文件列表中

4.4 文件预览流程
1. 在文件列表中找到要预览的文件
2. 点击文件名称或文件图标
3. 文件预览窗口将自动打开
4. 支持预览的文件类型：
   - 图片：JPG, PNG, GIF, BMP, WEBP, SVG等
   - 视频：MP4, AVI, MOV, MKV等
   - 音频：MP3, WAV, AAC, FLAC等
   - PDF文档
5. 在预览窗口中可以点击"下载"按钮下载文件
6. 点击右上角关闭按钮或按ESC键关闭预览

4.5 文件下载流程
方法一：从文件列表下载
1. 在文件列表中找到要下载的文件
2. 点击文件右侧的"下载"按钮
3. 文件将自动开始下载
4. 下载的文件保存在浏览器默认下载位置

方法二：从预览窗口下载
1. 打开文件预览
2. 点击预览窗口顶部的"下载"按钮
3. 文件将自动开始下载

4.6 文件删除流程
单个文件删除：
1. 在文件列表中找到要删除的文件
2. 点击文件右侧的"删除"按钮
3. 在确认对话框中点击"确认删除"
4. 文件将被移至回收站

批量文件删除：
1. 使用复选框选择多个文件
2. 点击顶部的"删除所选文件"按钮
3. 查看确认对话框中列出的文件列表
4. 点击"确认删除"执行删除操作

4.7 文件分享流程
单个文件分享：
1. 在文件列表中找到要分享的文件
2. 点击文件右侧的"分享"按钮
3. 在分享弹窗中选择分享方式
4. 复制分享链接或通过社交平台分享

批量文件分享：
1. 使用复选框选择多个文件
2. 点击顶部的"分享所选文件"按钮
3. 在分享弹窗中查看所有选中的文件
4. 选择分享方式并完成分享

4.8 文件重命名流程
1. 在文件列表中找到要重命名的文件
2. 点击文件右侧的"重命名"按钮
3. 在输入框中输入新的文件名
4. 按回车键或点击确认完成重命名

4.9 回收站操作
- 访问回收站：
1. 点击左侧边栏的"回收站"
2. 查看所有已删除的文件

- 恢复文件：
1. 在回收站中找到要恢复的文件
2. 点击文件右侧的"恢复"按钮
3. 文件将恢复到原位置

- 永久删除文件：
1. 在回收站中找到要永久删除的文件
2. 点击文件右侧的"删除"按钮
3. 在确认对话框中点击"确认永久删除"
4. 文件将被永久删除，无法恢复

五、常见问题解答
========================================

5.1 上传失败怎么办？
- 检查网络连接是否稳定
- 确认文件大小是否适合存储方式
- 小文件检查localStorage配额是否已满
- 大文件检查MEGA连接是否正常
- 尝试使用其他浏览器
- 确认存储空间是否已满

5.2 MEGA上传失败怎么办？
- 确保网络连接稳定
- 检查MEGA服务是否正常
- 尝试重新上传
- 如持续失败可联系技术支持

5.3 下载速度慢怎么办？
- 检查网络连接
- 尝试在网络空闲时段下载
- 大文件建议使用下载工具
- MEGA文件下载速度取决于网络状况

5.4 忘记密码怎么办？
- 当前版本使用预设账户
- 请使用管理员账户登录
- 后续版本将推出密码重置功能

5.5 文件找不到怎么办？
- 检查是否在正确的文件夹中
- 查看回收站
- 使用搜索功能查找

5.6 文件无法预览怎么办？
- 确认文件已正确上传（重新上传试试）
- 检查文件类型是否支持预览
- 某些文件类型（如Word、Excel）需要下载后查看
- 图片、视频、音频和PDF文件支持预览
- MEGA文件预览需要网络连接

5.7 如何区分MEGA文件和本地文件？
- 用户界面显示一致，无需区分
- 系统内部通过isMegaFile属性标记
- 操作方式完全相同
- 下载、预览、删除等功能兼容

5.8 如何联系技术支持？
- 通过"设置"页面联系管理员
- 后续版本将推出在线客服功能
