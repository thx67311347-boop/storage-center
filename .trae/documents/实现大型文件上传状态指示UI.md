# 实现大型文件上传状态指示UI

## 现状分析
- 已有的上传进度显示功能基础
- 已实现MEGA上传进度回调
- 缺少预估剩余时间、取消按钮和超时处理

## 实现计划

### 1. 增强上传状态管理（FileManager.tsx）
- 添加上传开始时间记录
- 实现取消上传功能
- 添加上传超时检测
- 优化状态更新机制

### 2. 升级上传进度组件（UploadProgress.tsx）
- 添加预估剩余时间显示
- 添加取消上传按钮
- 增强视觉反馈（动画效果）
- 添加上传超时提示
- 优化移动端适配

### 3. 改进上传机制（useFileOperations.ts）
- 实现可取消的上传过程
- 添加上传超时处理
- 优化分块上传的稳定性
- 增强错误处理

### 4. MEGA上传优化（useMegaStorage.ts）
- 支持上传取消功能
- 增强进度回调的可靠性
- 添加上传超时检测

## 技术要点
- 使用React状态管理上传状态
- 实现基于进度的时间预估算法
- 添加上传取消的AbortController
- 实现上传超时检测机制
- 优化UI动画效果

## 预期效果
- 实时显示上传进度和预估剩余时间
- 提供取消上传的功能
- 上传超时时显示友好提示
- 稳定可靠的上传过程
- 符合应用整体风格的UI设计
