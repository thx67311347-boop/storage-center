一、文件上传规则
========================================

1.1 支持的文件格式
- 图片格式：JPG, JPEG, PNG, GIF, BMP, WEBP, SVG
- 文档格式：PDF, DOC, DOCX, XLS, XLSX, PPT, PPTX, TXT, MD
- 视频格式：MP4, AVI, MOV, MKV, WMV, FLV
- 音频格式：MP3, WAV, AAC, FLAC, OGG
- 压缩格式：ZIP, RAR, 7Z
- 其他格式：所有常见文件格式均支持

1.2 文件命名规范
- 文件名长度不超过255个字符
- 支持中文、英文、数字及常用符号
- 不建议使用特殊字符：\ / : * ? " < > |
- 建议使用有意义的文件名，便于查找和管理

1.3 上传方式
- 点击上传：点击"上传文件"按钮选择文件
- 拖拽上传：将文件拖拽到上传按钮区域
- 支持同时上传多个文件
- 上传按钮在拖拽时会有视觉反馈

1.4 最大单个文件大小限制
- 单个文件最大限制：100 MB
- 超过100 MB的文件将无法上传
- 系统会提示文件大小超限
- 建议大文件分卷压缩后上传

========================================

二、账户存储空间管理
========================================

2.1 存储空间限制
- 每个账户总存储空间：5 GB
- 存储空间包含所有文件（含回收站文件）
- 已删除文件在回收站中仍占用空间

2.2 已使用空间查询方法
- 登录后在左侧边栏查看存储空间使用情况
- 显示已使用空间和总空间百分比
- 彩色进度条直观展示空间使用状态

2.3 空间不足时的提示机制
- 当存储空间使用超过90%时，系统将显示警告提示
- 当存储空间使用超过95%时，系统将显示紧急提示
- 建议及时清理不需要的文件或考虑升级存储空间

2.4 扩容途径
- 当前版本暂不支持手动扩容
- 如需更多空间，请联系管理员
- 后续版本将推出付费扩容功能

三、操作指南与流程说明
========================================

3.1 账户注册
- 当前版本使用预设账户登录
- 管理员账户：admin@example.com
- 管理员密码：admin123
- 后续版本将开放用户自助注册功能

3.2 登录流程
1. 访问网站首页
2. 输入邮箱地址
3. 输入密码
4. 点击"登录"按钮
5. 登录成功后自动跳转至主页

3.3 文件上传流程
方法一：拖拽上传
1. 在主页空白处拖拽文件
2. 等待上传进度条完成
3. 文件自动显示在文件列表中

方法二：点击上传
1. 点击"上传文件"按钮
2. 在弹出的文件选择框中选择文件
3. 等待上传进度条完成
4. 文件自动显示在文件列表中

3.4 文件下载流程
1. 在文件列表中找到要下载的文件
2. 点击文件右侧的"下载"按钮
3. 文件将自动开始下载
4. 下载的文件保存在浏览器默认下载位置

3.5 文件删除流程
单个文件删除：
1. 在文件列表中找到要删除的文件
2. 点击文件右侧的"删除"按钮
3. 在确认对话框中点击"确认删除"
4. 文件将被移至回收站

批量文件删除：
1. 使用复选框选择多个文件
2. 点击顶部的"删除所选文件"按钮
3. 查看确认对话框中列出的文件列表
4. 点击"确认删除"执行删除操作

3.6 文件分享流程
单个文件分享：
1. 在文件列表中找到要分享的文件
2. 点击文件右侧的"分享"按钮
3. 在分享弹窗中选择分享方式
4. 复制分享链接或通过社交平台分享

批量文件分享：
1. 使用复选框选择多个文件
2. 点击顶部的"分享所选文件"按钮
3. 在分享弹窗中查看所有选中的文件
4. 选择分享方式并完成分享

3.7 文件重命名流程
1. 在文件列表中找到要重命名的文件
2. 点击文件右侧的"重命名"按钮
3. 在输入框中输入新的文件名
4. 按回车键或点击确认完成重命名

3.8 回收站操作
- 访问回收站：
1. 点击左侧边栏的"回收站"
2. 查看所有已删除的文件

- 恢复文件：
1. 在回收站中找到要恢复的文件
2. 点击文件右侧的"恢复"按钮
3. 文件将恢复到原位置

- 永久删除文件：
1. 在回收站中找到要永久删除的文件
2. 点击文件右侧的"删除"按钮
3. 在确认对话框中点击"确认永久删除"
4. 文件将被永久删除，无法恢复

四、常见问题解答
========================================

4.1 上传失败怎么办？
- 检查网络连接是否稳定
- 确认文件大小不超过100 MB
- 尝试使用其他浏览器
- 确认存储空间是否已满

4.2 下载速度慢怎么办？
- 检查网络连接
- 尝试在网络空闲时段下载
- 大文件建议使用下载工具

4.3 忘记密码怎么办？
- 当前版本使用预设账户
- 请使用管理员账户登录
- 后续版本将推出密码重置功能

4.4 文件找不到怎么办？
- 检查是否在正确的文件夹中
- 查看回收站
- 使用搜索功能查找

4.5 如何联系技术支持？
- 通过"设置"页面联系管理员
- 后续版本将推出在线客服功能
